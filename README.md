| In6            | 7         | P8\_11, GPIO              |
| In7            | 8         | P8\_14, GPIO              |
| In8            | 9         | P8\_16, GPIO              |
| In9            | 10        | P8\_15, GPIO              |
| In10           | 11        | P8\_18, GPIO              |

| In12           | 13        | P9\_15, GPIO              |
| In13           | 14        | P9\_23, GPIO              |
